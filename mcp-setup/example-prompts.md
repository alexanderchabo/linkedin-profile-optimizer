# LinkedIn MCP Example Prompts

Ready-to-use prompts for automating LinkedIn profile optimization with the MCP server.

## üéØ Profile Optimization

### Complete Profile Audit

```
Use the LinkedIn MCP to get my profile: https://www.linkedin.com/in/YOUR-PROFILE/

Then analyze it against these criteria:
1. Headline effectiveness (keywords, value prop)
2. About section (hook, storytelling, CTA)
3. Experience descriptions (results vs responsibilities)
4. Skills alignment with industry trends
5. SEO optimization

Provide a ranked list of improvements with specific rewrites for the top 3.
```

### Profile vs Job Comparison

```
1. Get my LinkedIn profile: [YOUR PROFILE URL]
2. Get this job posting: [JOB URL]
3. Create a gap analysis showing:
   - Skills in the job description I haven't listed
   - Keywords I should add to my profile
   - Experience highlights I should emphasize
   - Specific sentences I should add/modify
```

### Headline Optimization

```
Get my current LinkedIn profile and:
1. Extract my headline
2. Compare it to 5 top professionals in my field
3. Suggest 3 alternative headlines with reasoning
4. Each should be under 220 characters and include keywords
```

### Targeted Job Search

```

For the top 5 results:
1. Extract key requirements
2. Compare to my profile
3. Calculate a match score (%)
4. Suggest which one to apply to first

5. Suggested talking points for interview

- New York

Extract patterns in:
- Required skills (frequency analysis)
- Salary ranges
- Experience levels
- Emerging technologies mentioned
```

## üéØ Pro Tips

1. **Be Specific with URLs**: Always use full LinkedIn URLs for profiles, companies, and jobs
2. **Batch Requests**: Analyze multiple profiles/companies in one prompt for efficiency
3. **Regular Cadence**: Set reminders to run these prompts monthly for continuous optimization
4. **Save Results**: Keep a log of insights and track what you've implemented
5. **Privacy**: Never ask the MCP to interact/post - use it for read-only analysis
